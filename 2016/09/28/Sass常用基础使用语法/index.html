Sass常用的基础语法

    阅读之前最好有相应的Css（ps：会Css3更好）基础，Sass的Scss语法是Css3语法的一个超集，通过Sass编写代码可以用更少的代码量去完成更多的样式，除此之外更加便于代码的管理，也就是我们常说的代码模板化

  • 变量的声明
    Sass变量的格式一般如下：
    $color:blue;
    每一个变量的开头，都需要$符号，$后面跟着的是变量的名字，名字和将要赋值的属性值之间用：分割，这样就完成了一个变量的声明是不是很简单，声明结束后，所声明的变量默认是全局使用的
    特殊的变量声明：
    $image：image;
    background-#{$image}:url('');
    此时的$image不仅仅是作为一个属性值，直接的去拿去使用，而是$image本身也作为了属性的一部分，大家看清楚这里是作为属性而不是属性值，也就是说上面的background-#{$image}:url('');
    等价于background-image:url('');引入的方式为：#{$变量名字}

  • Sass中可以进行计算
    left:100px+200px;
    诸如此类的计算都是可以的，但是要注意运算符前后要以后一个空格的留白，不然计算会发生错误。

  • Sass中可以进行嵌套
    div{
        background:red;
            span{
                background:yellow;
                    }
            }
    上面的例子就是说一个div里面有span这个标签分别给这个div和span设置对应的背景颜色按照我们的惯性思维是这样的格式div span{ },但是sass中就可以用上面的这种嵌套格式（ps:关键看span的设置）

  • Sass中&的使用
    标准的Css代码：例如声明一个伪类选择器
    a:hover{display:none;}
    在Sass中就可以写成这样
    a{
    &:hover{display:none;} 

    }

    &可以代替一个父类的使用
